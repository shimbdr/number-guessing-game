# Number Guessing Game (C++)

This project is a simple C++ program that generates a random number between 0 and 100.  

The user tries to guess the number, and the program provides feedback if the guess is too high or too low until the correct number is found.

## Features

- Random number generation  
- User input handling  
- Feedback messages (higher/lower)  
- Loop-based guessing logic  

## Technologies Used

- C++
- Standard Libraries (cstdlib, ctime, iostream)
